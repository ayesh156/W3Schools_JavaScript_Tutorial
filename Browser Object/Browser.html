Navigator Object

Is cookies enabled in the visitor's browser?

The cookieEnabled property returns true if cookies are enabled

What is the name of the visitor's browser?

The appCodeName property returns the code name of the browser.

Do not reply on it! "Mozilla" is the application code name for Chrome, Firefox, IE, Safari and Opera.

What is the engine name of the visitor's browser?

The product property returns the product name of the browser.

Do not reply on it! Most browsers returns "Gecko" as product name!
